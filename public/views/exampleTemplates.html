Components guide

En principio necesitamos dos archivos, primero el template donde estará nuestro código html reutilizable que en este caso es box.hbs, y nuestro código javascript que en este caso es exampleTemplates.js.

Y obvio nuestro html principal, en este caso es exampleTemplates.html, pero ustedes crearán el html según la página, ya sea guardia.html, consulta.html, etc.

Ustedes utilizarán siempre el mismo template, el box.hbs, lo único que generarían por su cuenta es el js, que cambia según la página por estilos y contenido.

Acá les muestro cómo lo harían, echen un vistazo al JS.

¿Y si lo quiero de otros colores? ¿Incluso, que no sea para guardia, sino para consultas?

Como verán en el JS, la lógica es la misma.

Ahora falta otro componente, pero se renderizaría una vez, ¿cómo haríamos? Esta vez utilizaremos el template buttonAgregar.hbs, que también ustedes utilizarán.

Todos los hbs (templates) están para que ustedes lo utilicen.
